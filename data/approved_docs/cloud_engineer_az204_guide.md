## Certification Goal

AZ-204 is recommended for Cloud Engineers who need to design, build, test, and maintain cloud applications and services.

## Core Learning Areas

### Azure Functions

Learners should understand triggers, bindings, serverless execution patterns, error handling, and monitoring.

### API Development

Learners should understand REST API design, request handling, authentication patterns, and integration with cloud services.

### Azure Storage

Learners should understand blob storage, queues, tables, access control, and storage performance considerations.

### Authentication and Authorization

Learners should understand managed identities, OAuth concepts, token-based authentication, and secure access to resources.

### Monitoring and Troubleshooting

Learners should understand application logs, metrics, alerts, and basic troubleshooting workflows.

## Recommended Study Sequence

1. Review cloud application fundamentals.
2. Study Azure Functions.
3. Practice API development scenarios.
4. Study Azure Storage patterns.
5. Review identity and access patterns.
6. Complete a readiness assessment.

## Readiness Guidance

A learner should not be marked ready until they consistently score at least 75% on grounded practice questions and have covered all core learning areas.

## Example Grounded Practice Question Pattern

Question topics should be based on one or more core learning areas. Questions should test reasoning, not memorization.
